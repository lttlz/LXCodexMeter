# LX Codex Meter v0.6.16

这是一个窗口交互修复版本。

## 修复内容

- 修复悬浮窗或任务条贴近屏幕底部时，点击设置后设置页无法正常弹出的问题。
- 设置页现在会以原窗口底边为锚点向上展开，并保持在当前显示器工作区内。
- 修复关闭设置后窗口尺寸或位置未正确恢复的问题。
- 改进多显示器、负坐标和 Windows 高 DPI 缩放环境下的窗口定位。
- 设置内容高度超过屏幕工作区时支持滚动访问。

## 安全与隐私

LX Codex Meter 仍只通过：

`codex.exe app-server --stdio`

读取官方额度信息。

不会读取浏览器 Cookie、Token、auth.json、对话内容或其他隐私数据。
