LX Codex Meter v0.6.12

LX Codex Meter v0.6.12 updates the official application icon with a clearer background-based design for better visibility in the Windows tray, Start menu, installer, and small-size icon contexts. It also adds support and contact information at the bottom of the settings page. The app remains fully free, with no paid restrictions and no mandatory donation. This release also adds a “Check for Updates” button and improves the stability of the taskbar strip mode when kept on top.

Privacy:
LX Codex Meter does not read browser cookies, tokens, auth.json, web pages, user projects, or Codex conversation content. Usage data is retrieved only through the local Codex App Server. The automatic update feature only contacts the official GitHub / Gitee release endpoints and collects no user data.
