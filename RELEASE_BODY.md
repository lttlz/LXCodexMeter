LX Codex Meter v0.6.11

LX Codex Meter v0.6.11 是一个正式整理版本，主要面向 Windows 用户，通过 Codex App Server 自动读取并显示 Codex 额度状态。

更新内容：
- 新增正式应用图标
- 提供 Windows 安装包
- 支持默认悬浮窗
- 支持任务栏条模式
- 支持系统托盘菜单
- 支持 Codex App Server 自动实时读取
- 显示 5 小时额度、周额度、重置时间和 Credits 信息
- 整理项目链接与基础发布信息
- 接入 Tauri Updater 自动更新（签名校验）

隐私说明：
本工具不读取浏览器 Cookie、token、auth.json，不抓取网页，不扫描用户项目，不读取 Codex 会话内容。额度信息仅通过本机 Codex App Server 获取。自动更新仅连接官方 GitHub / Gitee 发布源，不收集任何用户数据。

LX Codex Meter v0.6.11 is a polished Windows release that displays Codex usage status through Codex App Server.

Changes:
- Added the official application icon
- Added Windows installer packages
- Added compact floating window support
- Added taskbar strip mode
- Added system tray integration
- Added automatic Codex App Server usage reading
- Displays 5-hour usage, weekly usage, reset times, and Credits when available
- Updated project links and release metadata
- Integrated Tauri Updater for automatic updates (signature verified)

Privacy:
LX Codex Meter does not read browser cookies, tokens, auth.json, web pages, user projects, or Codex conversation content. Usage data is retrieved only through the local Codex App Server. The automatic update feature only contacts the official GitHub / Gitee release endpoints and collects no user data.
