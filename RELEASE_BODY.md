LX Codex Meter v0.6.14

LX Codex Meter v0.6.14 improves startup behavior, Codex visibility sync, taskbar strip display, update installation, localization, and installer branding. This release adds optional Windows autostart, optional start-hidden behavior, optional auto-show when Codex is running, and optional auto-hide when Codex closes. Taskbar strip mode now shows reset voucher count and uses “Data” for the latest data read time. It also strengthens topmost keep-alive behavior, preserves user-adjusted window width when opening or closing settings, adds Chinese/English language switching, improves the check-update flow with user-triggered download and install, and unifies app and installer icons. Data source and privacy behavior remain unchanged.

Privacy:
LX Codex Meter does not read browser cookies, tokens, auth.json, web pages, user projects, or Codex conversation content. Usage data is retrieved only through the local Codex App Server. The Codex sync feature only checks whether the local codex.exe process is running; it never reads Codex files, sessions, or credentials. The automatic update feature only contacts the official GitHub / Gitee release endpoints and collects no user data.
